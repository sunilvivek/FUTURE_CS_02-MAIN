# Phishing Email Detection & Awareness System

## Overview

This project was completed as part of my Cyber Security Internship at Future Interns.

The goal of this task was to analyze phishing emails, identify common phishing indicators, classify email risks, and create awareness materials that help users recognize and avoid phishing attacks.

Through this project, I learned how attackers use social engineering techniques such as urgency, fear, and deceptive links to trick users into revealing sensitive information.

---

## Objectives

1. Analyze phishing email samples.
2. Identify phishing indicators.
3. Differentiate between legitimate and malicious emails.
4. Classify email risk levels.
5. Create user awareness and prevention guidelines.
6. Promote safe email practices.

---

## Email Analysis Performed

### Phishing Email Analysis

The phishing email was examined for:

1. Suspicious sender domain.
2. Urgent subject line.
3. Generic greeting.
4. Suspicious verification link.
5. Fear-based language.

**Classification:** Phishing

**Risk Level:** High

### Legitimate Email Analysis

A legitimate email sample was also analyzed to understand the characteristics of genuine communication.

Key observations included:

1. Trusted sender domain.
2. Personalized greeting.
3. Professional communication style.
4. No request for sensitive information.

**Classification:** Safe

**Risk Level:** Low
